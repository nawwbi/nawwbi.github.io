notes application

This is an application that uses 'no-code'. 'Notes' is an app where users can add new notes and write detailed descrptions under it.

Other than that, users can also delete the notes and edit them to their liking.

There is also another tab available called the 'Favourites', where users can add their important notes and quickly access them from the navigation bar on the left.

"suggestAI"

This application was made using 'low-code', and utilizes OpenAI. According to the questions that the user asks with the search bar, this application would provide the top 5 best answers for that question.

This application also allows users to place their asked questions and answers into a 'Favourites' to allow easy access. Users can simple click on one of the suggestions and click on the 'Favourites' button.

Application: 'Glide'
Language(s): Java

SOI Learning Portal

A project made for a module in school. This is an online website for SOI's learning Portal.

Users are able to find out more on courses they are interested in by clicking on the 'learn more' button. They can also place them into their off-canvas bookmarks by clicking on the bookmark icon at the bottom right of the card.

Software: VSCode
Language(s): HTML, Bootstrap, jQuery

A Movie List

A school project utilising Android Studio to produce a movie list. Users are able to insert the Title, Genre, Year produced, and Age rating of the movie they would like to add into the list.

'Show List' button shows the list of movies that were inserted, together with the other details.

Users are able to edit and delete the contents of the movie list by simply clicking one of the movies. Lastly, a filter function located at the top of the list page allows users to filter between all movies and PG-13 movies.

Software: Android Studio
Language(s): Java

A Song List

Another school project produced in Android Studio, a list of songs rated by the user. Users are able to insert the song's Title, Singer(s), Year of production, and their rating out of 5 stars.

The 'Show List' button displays the list of songs that were inserted as well as its details. Users are also allowed to edit its contents by clicking on any song in the list and updating it after.

The button at the top of the list works as a filter for songs that were rated with 5 stars.

Software: Android Studio
Language(s): Java
